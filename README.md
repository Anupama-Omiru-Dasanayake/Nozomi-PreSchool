# Nozomi Pre School Website

A responsive informational website designed for **Nozomi Pre School** to present school information, programs, and activities to parents and visitors.

The project focuses on creating a clean and user-friendly interface with clear navigation so parents can easily access important information about the preschool.

## Features

- Responsive website design
- School information and program details
- Photo gallery for school activities
- Contact information for parents
- Simple and clean user interface

## Technologies Used

- HTML
- CSS
- Java Script

## Purpose

The purpose of this project is to create a simple, accessible, and visually appealing website for a preschool to improve communication with parents and showcase school activities.
